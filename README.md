# Control of an Endoscope Robot for Ear and Nose Surgery

## Description
This project contains kinematics code for a spherical robot used in endoscopic surgical applications.

## Files
- firstdraft_dk_numeric.m → Direct kinematics
- firstdraft_ik_complete.m → Inverse kinematics
- firstdraft_jacobian_runner.m → Jacobian testing
- mechanism_jacobian.m → Jacobian computation
